json格式化性能优化

Posted on 2020-03-06

  1. 语言：python

  2. 场景：由于history接口返回图片是base64编码，当limit较大时返回大json字符串。

  3. 测试方式：在入口文件加入Profiler

  4. 测试结果：

可见在 return jsonify({…})时候，simplejson encode耗时比较久，30ms左右。

  1. 改进：使用ujson，并覆盖jsonify方法

性能提升大概2倍，耗时减少至15ms左右.
